# ✅ Solução Simples - Como Usar o Aplicativo

## 🎯 Opção Mais Simples: Usar o Executável Electron Diretamente

O aplicativo já tem Electron configurado. Para usar sem scripts complexos:

### Passo 1: Instalar Dependências (Uma Única Vez)

Abra o PowerShell ou CMD na pasta do projeto e execute:

```powershell
pnpm install
```

### Passo 2: Compilar o Servidor (Uma Única Vez)

```powershell
pnpm build
```

### Passo 3: Usar o Aplicativo

**Duplo clique em:** `INICIAR.bat`

OU execute diretamente:

```powershell
pnpm electron:start
```

O aplicativo abrirá em uma janela do Windows automaticamente!

## 🌐 Opção Alternativa: Usar Versão Web Online

## 📱 Opção Mobile: App Expo Go

1. Instale o Expo Go no celular
2. Execute: `pnpm dev`
3. Escaneie o QR code
