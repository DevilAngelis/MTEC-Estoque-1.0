COMO USAR O APLICATIVO
═══════════════════════════════════════════════════════════════

OPÇÃO 1: USO NORMAL (RECOMENDADO)
───────────────────────────────────────────────────────────────

1. Dê duplo clique no arquivo:
   📄 INICIAR_APLICATIVO.bat

2. Aguarde alguns segundos enquanto o aplicativo inicia

3. O navegador abrirá automaticamente com o aplicativo

4. Use o aplicativo normalmente!


OPÇÃO 2: MODO DESENVOLVIMENTO
───────────────────────────────────────────────────────────────

1. Dê duplo clique no arquivo:
   📄 INICIAR_DESENVOLVIMENTO.bat

2. Use quando quiser testar mudanças no código


REQUISITOS
═══════════════════════════════════════════════════════════════

- Windows 10 ou superior
- Node.js instalado (baixe em: https://nodejs.org)
  Escolha a versão LTS (Long Term Support)

O aplicativo instalará automaticamente:
- pnpm (gerenciador de pacotes)
- Todas as dependências necessárias


PRIMEIRA VEZ
═══════════════════════════════════════════════════════════════

Na primeira execução, o aplicativo irá:
✓ Instalar pnpm (se necessário)
✓ Instalar todas as dependências
✓ Compilar o servidor
✓ Configurar o banco de dados

Isso pode levar alguns minutos. Nas próximas vezes será muito mais rápido!


COMO FECHAR
═══════════════════════════════════════════════════════════════

- Feche a janela do navegador
- Feche a janela do terminal (ou pressione Ctrl+C)


PROBLEMAS?
═══════════════════════════════════════════════════════════════

Se o aplicativo não iniciar:

1. Verifique se o Node.js está instalado:
   - Abra o Prompt de Comando
   - Digite: node --version
   - Se não aparecer uma versão, instale o Node.js

2. Verifique se a porta 3000 está livre:
   - Feche outros aplicativos que possam estar usando a porta

3. Execute como Administrador:
   - Clique com botão direito no arquivo .bat
   - Selecione "Executar como administrador"


LOCALIZAÇÃO DOS DADOS
═══════════════════════════════════════════════════════════════

O banco de dados está localizado em:
%APPDATA%\MTec Estoque\database

Para fazer backup, copie esta pasta.
