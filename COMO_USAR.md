# Como Usar o MTec Estoque

## Forma mais simples (sem instalar nada)

Abra o navegador e acesse:

**https://mtec-estoque-1-0-production.up.railway.app**

Pronto. Não precisa de script nem de comando.

---

## Se quiser rodar no seu PC

No terminal (PowerShell ou CMD), na pasta do projeto:

1. Instalar dependências (uma vez):
   ```
   pnpm install
   ```

2. Compilar (uma vez):
   ```
   pnpm build
   ```

3. Subir o servidor:
   ```
   pnpm start
   ```

4. Abrir no navegador: **http://localhost:3000**

---

**Recomendação:** use o link online acima; não depende de script nem de nada no seu PC.
